# Teddy Trading Bot (Bitsure Teddy)

Bot Telegram professionnel d'analyse technique pour crypto, forex, actions et matières premières.

## Déploiement Railway

1. Forkez ce dépôt.
2. Connectez-le à Railway.
3. Ajoutez les variables d'environnement :
   - `TELEGRAM_TOKEN`
   - `FCS_API_KEY`
   - `REALMARKET_API_KEY`
4. Railway détectera le Procfile et lancera le bot.

## Commandes

Utilisez `/help` dans Telegram pour la liste complète.
